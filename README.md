# layoffs-data-cleaning-eda

## Overview
This project explores global layoffs from 2020 to 2023. The data was cleaned 
and analysed using SQL to uncover trends and patterns in layoffs across 
different companies, industries, and countries.

## Data Source
The dataset was sourced from GitHub and covers global layoffs from 2020 to 2023.

## Files
| File | Description |
|------|-------------|
| [data_cleaning.sql](SQL_Data_Cleaning.sql) | All data cleaning steps |
| [exploratory_data_analysis.sql](Exploratory_Data_Analysis.sql) | All EDA queries |

## Data Cleaning Steps
- Removed duplicate records
- Handled null and blank values
- Standardized formats and values
- Validated the final cleaned data

## Exploratory Data Analysis
- Summary statistics on layoffs
- Layoffs by company, industry and country
- Layoffs trends over time (2020-2023)

## Tools Used
- MySQL
